# DG App UI

Front-end da aplicação DGApp desenvolvida com Vuerify (Vue.js + Material Design)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
